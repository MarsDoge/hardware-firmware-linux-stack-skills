# Lab Checklists

## Pre-Power
- Visual inspection
- Board revision/BOM verified
- Rail resistance-to-ground
- Current limit configured
- Boot straps documented
- Debug headers identified
- Recovery path available

## First Power
- VIN applied with current limit
- Input current recorded
- Always-on and main rails measured
- Reset and power-good captured
- Clocks verified
- Thermal check performed

## Boot
- UART/JTAG connected
- Boot mode confirmed
- Boot source present
- Firmware hash recorded
- Full serial log captured
- Milestones updated

## Handoff
- Evidence folder complete
- Issues labeled by layer
- Reproduction steps written
- Next measurements defined
