# USB Type-C / PD Evidence Checklist

- [ ] Port topology and controller/PHY/TCPC/mux/retimer parts
- [ ] VBUS/VCONN/CC measurements
- [ ] Cable/device/charger identity
- [ ] PD trace or tcpm/ucsi logs
- [ ] `dmesg`, `lsusb`, `lsusb -t`
- [ ] DT/ACPI resources
- [ ] USB2 vs USB3/USB4 behavior separated
