# DDR/LPDDR Workflows

## Training failure
Collect firmware serial log, board rev/BOM, memory part, topology, PMIC/rail measurements, clock/reset timing, SPD/straps, training stage, and known-good comparison.

## Runtime instability
Collect dmesg, EDAC/MCE/RAS logs, stress result, temperature/load condition, DIMM slot/channel/rank map, BIOS memory settings, ECC status, and firmware version.
